macOS AI Terminal Agent
A lightweight assistant that converts natural language into ZSH commands using local LLMs via Ollama. It includes built-in safety filters to prevent accidental data loss.

Prerequisites
macOS: This tool is designed specifically for ZSH on Apple systems.

Ollama: Download and install from ollama.com.

Llama3 Model: Pull the model by running: ollama pull llama3

Trash Utility: Required for safety features. Install via Homebrew: brew install trash

Installation
Clone this repository:
git clone https://github.com/yourusername/macos-ai-agent.git

Enter the directory:
cd macos-ai-agent

Install Python dependencies:
pip install ollama

Set up an alias (optional):
Add this line to your ~/.zshrc file to run the agent by typing 'ai':
alias ai='python3 /absolute/path/to/your/agent.py'

Refresh your terminal:
source ~/.zshrc

Usage
Run the script by typing 'ai' or 'python3 agent.py'. Type your request in plain English when prompted.

Example prompts:

Open my downloads folder

Move all png files from desktop to a new folder called images

Show my local ip address

Delete the file named old_notes

Safety Features
All rm commands are automatically converted to trash to allow file recovery.

The script includes a forbidden keyword filter for destructive actions.

You must manually confirm every command with (y/n) before it executes.

It automatically appends wildcards (*) to filenames without extensions.

Disclaimer
